# 个人总结

一些书籍：

《Rust编程：入门 实战与进阶》（朱春雷）：适合入门，但貌似没有讲宏

《Rust第一课》（陈天）：juejin.cn网站买的课程，上面一本书学完，再看这个

《Rust程序设计（第2版）》：吉姆·布兰迪，rust程序员人手必备

《张汉东的 Rust 实战课》：极客时间，视频需要会员，68元

《Rust编程之道》（张汉东）：非入门书籍，建议放到后面学

《Rust Web开发》：计划阅读

《Rust并发编程实战》：计划阅读

《Rust死灵书》：计划阅读
