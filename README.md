# 个人总结

一些书籍：

《Rust编程：入门 实战与进阶》（朱春雷）：很多人批评这本书简单，但对我来说帮助非常大，只需要一些c语言基础即可

《Rust第一课》（陈天）：juejin.cn网站买的课程，上面一本书学完看这个非常好

《Rust程序设计（第2版）》：吉姆·布兰迪，rust程序员人手必备

《Rust编程之道》（张汉东）：非入门书籍，建议放到后面学

《Rust Web开发》：计划阅读

《Rust并发编程实战》：计划阅读
